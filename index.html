How It Works

The Point System

Giveaway: The user that gives away a free toy gets +1 point

Receive: The user that receives a free toy gets -1 point

Exchange: Both parties get 0 points

App Features

  • Transaction history
  • User ratings/reviews
  • Filters based on toy category, age range, request, and user rating
  • Sort based on age range and user rating
